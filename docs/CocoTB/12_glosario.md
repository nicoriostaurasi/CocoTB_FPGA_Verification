# Glosario

## cocotb

Framework para escribir testbenches en Python y conectarlos a simuladores HDL.

## DUT

Device Under Test. Módulo que se quiere verificar.

## HDL

Hardware Description Language. Lenguaje para describir hardware, como Verilog o SystemVerilog.

## Icarus Verilog

Simulador open source usado para compilar y correr los ejercicios.

## Trigger

Evento que una corrutina espera para continuar. Ejemplos: `Timer`, `RisingEdge`, `FallingEdge`, `ClockCycles`.

## Corrutina

Función asincrónica que puede pausarse y reanudarse con `await`.

## Testbench

Código que aplica estímulos al DUT y verifica respuestas.

## Waveform

Representación temporal de señales digitales.

## VCD

Value Change Dump. Archivo de ondas generado por la simulación.

## Scoreboard

Modelo de chequeo que compara salidas observadas contra valores esperados.

## Reset

Señal que lleva el DUT a un estado conocido.

## Clock

Señal periódica que sincroniza lógica secuencial.
